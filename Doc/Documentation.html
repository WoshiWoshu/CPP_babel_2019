Project presentation



The Babel is a 3rd year Epitech project from the module Advanced C++. It consist in a VOIP server / client programm (like Skype). The main goal of the project is to discover deeper C++ architecture. The project has to build and run the same on Unix and Windows systems. Creating software abstractions that allow natively portable programs to be developed on all of the systems. This abstraction focuses on a key concept that governs the entirety of our course's themes: Application Programming Interfaces (API), which are elements that are inseparable from Object-Oriented Programming, all languages combined.


VOIP (Voice over Intert Protocol)


Voice over Internet Protocol is a system that transforms analog audio signals into digital data that can be transmitted over the Internet. A VoIP client cannot work alone. To be able to make the calls, it has to work with a VoIP service or a SIP server. The Session Initiation Protocol (SIP) is a signaling protocol used for initiating, maintaining, and terminating real-time
sessions that include voice, video and messaging applications.


CMAKE


The project compilation has to held with CMake. This is a cross-platform meta build system for managing the build process of software using a compiler-independent method. It uses scripts called CMakeLists to generate build files< for a specific environment (for example, Makefiles on Unix machines).


CONAN


Conan is C / C++ open source multi-platform package manager for developers. It provides creation and sharing binaries; enables users to encapsulate C/C++ project dependencies, distribute them, and consume them in other projects.


PORT AUDIO


PortAudio is a C open-source computer library for audio playback and recording. It is a cross-platform library, so programs using it can run on many different computer operating systems, including Windows, Mac OS X and Linux.


QT Creator


The graphical user interface has to be made with Qt Creator. It is an IDE designed with Qt to develop projects, including various tools and allows:

  • Project management (creation of pre-filled
  • Compilation and execution of applications
  • Creation of windows and widgets using Qt tools
  • Version management (Git, Subversion, Bazaar, CVS...)
It also makes it easier to create and compile laptop applications.
